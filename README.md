# OSModule

[![NPM](https://img.shields.io/npm/l/react)](https://github.com/lgfvarella/OSModule/blob/main/LICENSE)

# Sobre o Projeto

OSModule é um WebApp para fazer a emissão e gestão de ordens de serviço.

O App tem os seguintes recursos:
- Cadastro de Clientes
- Cadastro de Técnicos
- Emissão de Ordens de Serviço
- Status de Ordens de Serviço
- Baixa de Ordens de Serviço

# Tecnologias utilizadas

## Back end
- Java 17
- Spring 4.0
- MySQL
- Lombok
- Maven
